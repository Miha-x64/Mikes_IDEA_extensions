Reports use of XML attributes or attribute values which are not supported in current minSdk.

<bitmap android:tint>

Available since Lollipop.

<include layout="?attr">

LayoutInflater resolves the value of <include>'s layout attribute correctly only since this commit, Android 6.0.1 Marshmallow.

On earlier versions you may use <ViewStub> as a workaround.

Given that XML layouts are super crappy and not mature enough even after a decade of Android evolution, I personally recommend you a global fix: throw them away and use Splitties View DSL, or roll your own UI DSL in one hour and 100 LoC which will suit your needs.

The inspection is provided by Mike's IDEA Extensions.
